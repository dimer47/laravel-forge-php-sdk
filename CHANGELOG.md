## [1.0.0] - 2025-01-24

### Added
- Initial release
- Complete Laravel Forge API coverage (245 endpoints)
- 28 API classes for all Forge resources
- 167 typed models for requests and responses
- Full documentation and usage examples
- Support for PHP 8.1+
